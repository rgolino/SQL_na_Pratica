
# 📘 SQL na Prática

Bem-vindo ao repositório **SQL na Prática**! Aqui você encontrará uma abordagem prática e descomplicada para aprender SQL, a linguagem padrão para comunicação com bancos de dados relacionais.

---

## 📌 O que é SQL?

**SQL (Structured Query Language)** é uma linguagem usada para **consultar, inserir, atualizar e excluir dados** em sistemas de gerenciamento de banco de dados relacionais (como MySQL, PostgreSQL, SQL Server, SQLite, entre outros).

---

## 🎯 Objetivo do Projeto

Este projeto tem como principal objetivo **ensinar SQL de forma prática**, com exemplos reais de negócios, desafios interativos e exercícios para consolidar o aprendizado. Ideal para iniciantes e também para profissionais que desejam reforçar seus conhecimentos com foco no uso prático da linguagem.

---

## 🧠 O que você vai aprender

- Conceitos básicos de banco de dados relacionais  
- Criação de tabelas (DDL - Data Definition Language)  
- Inserção, consulta, atualização e exclusão de dados (DML - Data Manipulation Language)  
- Filtros, ordenações e funções agregadas  
- Joins (INNER, LEFT, RIGHT, FULL)  
- Subqueries e CTEs (Common Table Expressions)  
- Agrupamentos e agrupamentos avançados (GROUP BY, HAVING)  
- Criação de views e procedures  
- Boas práticas em SQL  
- Casos de uso do mundo real

## 🛠 Pré-requisitos

Para executar os scripts SQL, você pode usar:

- [DBeaver](https://dbeaver.io/) ou [DB Browser for SQLite](https://sqlitebrowser.org/)
- Um servidor local como MySQL, PostgreSQL ou SQLite
- Alternativamente, use o [SQLite Online](https://sqliteonline.com/) para testar rapidamente

## 🚀 Como usar

1. Clone o repositório:
```bash
git clone https://github.com/seuusuario/sql-na-pratica.git
```

2. Abra os arquivos `.sql` na sua ferramenta preferida

3. Siga as instruções e comentários em cada script para praticar e testar os comandos
